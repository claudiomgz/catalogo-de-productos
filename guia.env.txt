Variables de entorno a definir
=======================================
EMAIL_USER: usuario@gmail.com
EMAIL_PASS: 12345
EMAIL_RECIBE: El mail que va a recibir los mails de contacto
DB_USER: usuario de base de datos
DB_DB: La base de datos a usar
DB_HOST: el dominio donde se aloja la base de datos
DB_PASS: la clave de la base de datos
SESSION_SECRET: clave secreta
